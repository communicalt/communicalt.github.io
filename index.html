Software de comunicação alternativa para crianças com deficiência

SAIBA MAIS

Nosso projeto

Desenvolvemos um software de criação de pranchetas de comunicação alternativa para crianças com deficiência, que ajuda os profissionais especializados nessa área, tanto para o desenvolvimento da criança, quanto o acompanhamento do processo de aprendizagem.

Além de criar pranchetas, o software permite a criação de símbolos,a exportação (de símbolos e pranchetas) e a utilização de recursos auditivos implementados nos símbolos para manter a atenção das crianças na rotina de AEE (Atendimento Educacional Especializado).
